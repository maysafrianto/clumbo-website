Selamat Datang di Clumbo Website

Clumbo Website merupakan sebuah platform web statis yang dirancang khusus untuk memenuhi tugas Ujian Akhir Semester (UAS). Proyek ini berfokus pada implementasi arsitektur web dasar yang bersih, responsif, dan siap dideploy ke lingkungan server.

Melalui proyek ini, seluruh komponen kode dikelola menggunakan version control system (Git) dan diunggah ke platform GitHub untuk mempermudah integrasi dengan server VPS.

Teknologi & Fitur

Meskipun dibangun menggunakan arsitektur dasar, Clumbo Website mengedepankan beberapa prinsip modern:

Responsive Design

Tampilan website otomatis menyesuaikan dengan ukuran layar perangkat (HP, Tablet, maupun Laptop).

Clean Code

Struktur penulisan HTML5 dan CSS3 yang rapi dan terorganisir dengan baik sehingga mudah dirawat.

VPS & Cloud Ready

Konfigurasi direktori file telah dioptimalkan agar siap dijalankan menggunakan web server seperti Nginx atau Apache.
